### HOW TO UPDATE？

1. Download the zip format file and unzip it before uploading.
2. Access the web console of the device, and go to "System" -- "Firmware Update".
3. Upload the bin file, and wait to finish.

Good luck, check the model before update, This repo is for ESP32 based model: GeekMagic SmallTV Pro.
